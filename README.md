# alimenta Aí!

Projeto alinhado ao ODS 2 (Fome Zero), focado em reduzir o desperdício de alimentos em comércios. Propõe um sistema web que conecta doadores a pessoas ou entidades em insegurança alimentar, com gestão de usuários e dados em tempo real, garantindo distribuição eficiente e segura dos alimentos.

---
## Requisitos Funcionais
#### Cadastro de usuário:
O sistema deve permitir que usuários realizem cadastro informando dados básicos como nome, email e senha.
#### Autenticação de Usuário:
O sistema deve permitir fazer login utilozando email e senha.
#### Cadastro de Doações:
O sistema deve permitir que usuários registrem alimentos disponíveis para doação.
#### Listagem de Doações:
O sistema deve exibir uma lista de alimentos disponíveis para doação.
#### Solicitação de Doações:
O sistema deve permitir que usuários solicitem alimentos disponíveis.
#### Controle de Status:
O sistema deve permitir alterar o status das doações (disponível, reservado, entregue).

---
## Requisitos Não Funcionais
#### Usabilidade:
O sistema deve possuir interface simples e intuitiva.
#### Desempenho:
O sistema deve responder às requisições em tempo adequado.
#### Segurança:
Os dados dos usuários devem ser protegidos.
#### Responsividade:
O sistema deve funcionar em diferentes tamanhos de tela (desktop e mobile).

---
## Atores do Sistema
- Usuário doador;
- Usuário receptor.
